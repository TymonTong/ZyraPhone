# 登录页背景图

当前目录已含 Figma 导出的临时预览图，你可直接替换为正式资源。请使用以下文件名：

| 文件名 | 对应设计稿图层 |
|--------|----------------|
| `bg.png` | bg（底层全屏背景） |
| `middle.png` | middle（中间装饰层） |
| `overlay.png` | overlay（顶层蒙版/渐变） |

可选：若需 Google 登录按钮品牌图标，可放置 `google.png`（14×14 左右）；未放置时页面会尝试使用 OT 图标库，若库中无对应 symbol 则隐藏左侧图标。

支持 `.webp` 时可在 `LoginPage.vue` 中把扩展名改为 `.webp`。
